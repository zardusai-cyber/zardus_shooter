> A cute, Ghibli-style top-down shooter featuring Baby Zardus! Defend against waves of adorable but mischievous enemies!

## 🌟 Features

- **Adorable Ghibli-style Graphics** - Hand-crafted SVG character of Baby Zardus
- **Smooth 60FPS Gameplay** - Pure Canvas rendering
- **Particle Effects** - Explosions, trails, muzzle flashes
- **Power-Up System** - Spread shot and double shot
- **Wave System** - Progressive difficulty
- **Audio Engine** - Procedural sound effects
- **High Score** - Local storage persistence
- **Mobile Support** - Touch controls for phones
- **Responsive Design** - Beautiful UI with glassmorphism

## 🎯 Gameplay

- **WASD** or **Arrow Keys** to move
- **SPACE** or **Click** to shoot
- **P** to pause
- Destroy enemies before they reach you!
- Collect golden stars for power-ups

## 🛠️ Technical Details

| Feature | Implementation |
|---------|----------------|
| Rendering | HTML5 Canvas 2D |
| Audio | Web Audio API (procedural) |
| Game Loop | requestAnimationFrame (60fps) |
| Particles | Custom particle system |
| Input | Keyboard + Mouse + Touch |
| Storage | localStorage (high scores) |
